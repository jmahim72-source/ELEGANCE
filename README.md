## 🎨 Design Philosophy
- **Quiet Luxury Palette:** Organic bone off-white canvas (`#F5F3EF`), sharp near-black text (`#161513`), and Ink Blue accents (`#2B3A4A`).
- **Typography:** Display headlines in high-contrast `Bodoni Moda` serif paired with neutral `Inter` body text.
- **Architectural Geometry:** Enforced 0px border-radius across all containers, inputs, and image frames.
- **Micro-Interactions:** GSAP ScrollTrigger reveals, 3D card tilt on mouse move, hover image crossfades, and magnetic link underlines.

## 🛍️ Key Features
- **Integrated Single Page Views:** Homepage, Outerwear Collection (PLP), Product Detail Page (PDP with zoom lightbox), Brand Story, Journal Monograph, and Shopping Bag.
- **Interactive Cart & Drawer:** Stateful slide-over bag drawer, quantity steppers, promo code discount system (`SILENT10` for 10% off), and multi-currency switcher (`USD $`, `EUR €`, `GBP £`).
- **Checkout Modal Flow:** Seamless 2-column checkout with contact, shipping, payment, and order confirmation.
- **Journal Monograph:** High-fashion editorial essays with a full-screen reader modal.
